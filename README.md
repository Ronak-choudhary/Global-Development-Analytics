# 🌍 Global Development Index: A Data-Driven Analysis of Socioeconomic Indicators

Analyze and compare countries using socioeconomic, governance, environmental, and human development indicators through data preprocessing, normalization, Principal Component Analysis (PCA), and visualization.

---

## 📌 Project Overview

Governments and international organizations collect hundreds of indicators to measure a country's development. However, comparing countries directly is challenging because these indicators differ significantly in scale, units, and importance.

This project builds an end-to-end analytics pipeline that transforms raw development indicators into a standardized dataset, applies dimensionality reduction, and generates meaningful insights into global development patterns.

---

## 🎯 Objectives

- Collect and integrate development indicators
- Clean and preprocess real-world datasets
- Handle missing and inconsistent values
- Normalize heterogeneous indicators
- Perform dimensionality reduction using PCA
- Visualize development patterns across countries
- Compare countries using a composite development framework

---

## 📊 Dataset

The project uses country-level development indicators covering multiple domains such as:

- Economic Performance
- Education
- Healthcare
- Governance
- Environment
- Infrastructure
- Human Development

The final analytical dataset combines more than **70 development indicators** from multiple publicly available international sources.
